## 🔬 In-Depth Analysis of Scores

### 1. Faithfulness (V1: 0.9243 vs V2: 0.6389)
* **Observation:** Prompt V1 (Concise) drastically outperforms V2 in Faithfulness.
* **Explanation:** 
  * **V1 System Prompt** strictly instructs the model: *"Answer the user's question using ONLY the provided context. Keep your answer concise (2-4 sentences). If the context does not contain the answer, say: 'I don't have enough information.'"* 
  * **V2 System Prompt** instructs the model to act as an *"expert AI tutor"* and write a *"clear, well-organized answer (3-5 sentences)"*. 
  * Because V2 encourages structured, tutor-like formatting, the LLM naturally tends to adopt a helpful, explanatory tone. This explanation often includes logical extensions or minor general-knowledge assumptions that are *not* explicitly written in the provided narrow RAG context, leading to lower **Faithfulness** scores under RAGAS evaluation.

### 2. Answer Relevancy (V1: 0.9191 vs V2: 0.8970)
* **Observation:** V1 has slightly higher Answer Relevancy than V2.
* **Explanation:** Since V1's answers are concise and tightly constrained to the specific question asked, they avoid fluff or extra introductory/concluding tutor statements. This results in high semantic alignment with the original question.

### 3. Context Recall (V1: 1.0000 vs V2: 1.0000) & Precision (V1: 0.9633 vs V2: 0.9667)
* **Observation:** Both pipelines share nearly identical Context Recall and Context Precision scores.
* **Explanation:** This is fully expected because **both pipelines utilize the same underlying FAISS vector retriever** (`k=3`) and index. The retriever successfully returns 100% of the relevant passages required to answer the questions.
