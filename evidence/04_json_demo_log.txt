=======================================================
  JSON Formatting Demo
=======================================================

[Valid JSON] ✅ Pass
  Input:  {"name": "Alice", "age": 30}
  Output: {"name": "Alice", "age": 30}
  🔧 JSON repaired successfully

[Markdown fences] ✅ Pass
  Input:  ```json
{"name": "Bob"}
```
  Output: {
  "name": "Bob"
}
  🔧 JSON repaired successfully

[Single quotes] ✅ Pass
  Input:  {'name': 'Charlie', 'score': 95}
  Output: {
  "name": "Charlie",
  "score": 95
}
  🔧 JSON repaired successfully

[Trailing comma] ✅ Pass
  Input:  {"key": "value",}
  Output: {
  "key": "value"
}
  ❌ JSON repair failed, returning fallback JSON

[Truly invalid] ✅ Pass
  Input:  This is not JSON at all: ??? {]
  Output: {"error": "unparseable_json", "raw": "This is not JSON at all: ??? {]"}
